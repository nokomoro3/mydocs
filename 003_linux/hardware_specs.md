# ハードウェア情報確認

- OS
```sh
$ lsb_release -a
```

- カーネル
```sh
$ uname -a
```

- CPU
  - threadsがコア数と思えばよい。
```sh
$ sudo lshw -class processor
```

- GPU
```sh
$ lspci | grep -i nvidia
```

- RAM
```sh
$ sudo dmidecode -t memory
```

- マザーボード
```sh
$ sudo dmidecode -t baseboard
```
